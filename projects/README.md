# Projects

Проектные неймспейсы общего coordination hub.

Назначение:

- не смешивать артефакты разных проектов;
- сохранять общий workflow в одном месте;
- дать reviewer-агентам общий пишущий слой без доступа на запись в owner-only рабочие репозитории.

Текущие неймспейсы:

- `docspectrum/` - координация проекта `DocSpectrum`.

Checker-специфичные legacy-документы пока остаются в корневых `docs/`, `contracts/`, `discussion/` для сохранения ссылок.
