# DocSpectrum Discussion

Обсуждения по проекту `DocSpectrum` между generator/reviewer/human.

Рекомендуемый формат темы:

- `T-YYYYMMDD-NNN_slug.md`

Формат сообщений:

```text
## YYYY-MM-DD HH:MM — author

[TAG] Краткое содержание.

Evidence:
- ссылка на commit / файл / артефакт
```

Теги:

- `[FYI]`
- `[AGENT-OK]`
- `[NEEDS-HUMAN]`
- `[REVIEW]`
- `[FOLLOW-UP]`
