# Checker Shared Project

Общий координационный слой разработки Checker.

Назначение: фиксировать планы, исходные условия, граничные условия, контракты данных, метрики, решения и ссылки на реализации в рабочих репозиториях Codex / Opus / Fable.

Этот проект не заменяет рабочие репозитории участников. Он является общей диспетчерской и журналом трассировки решений.

## Папки

- `docs/` - состояние проекта, план, решения, вопросы, словарь.
- `contracts/` - контракты данных, метрик, причин и артефактов.
- `registries/` - реестры методов, прогонов, источников и репозиториев.
- `handoff/` - короткие передачи состояния между сессиями.

## Связанные общие папки

- `E:\commons\checker-shared-project` - канонические общие исходные, ссылки на ground truth и справочники.
- `E:\output\checker-shared-project` - результаты общих прогонов и отчеты.
