# Agents Coordination Hub

Общий координационный слой совместных проектов `human`, `codex`, `opus` и других агентов.

Назначение: фиксировать планы, исходные условия, граничные условия, контракты данных, метрики, решения, review packets и ссылки на реализации в рабочих репозиториях.

Этот проект не заменяет рабочие репозитории участников. Он является общей диспетчерской и журналом трассировки решений.

Исторически репозиторий был создан для Checker и сохраняет имя `checker-shared-project`. После `DEC-20260614-004` он используется как общий coordination hub для нескольких проектов. Корневые документы и существующие Checker-артефакты не переносятся, чтобы не ломать ссылки.

## Папки

- `docs/` - общие решения, состояние, план, вопросы, словарь.
- `contracts/` - общие контракты данных, метрик, причин и артефактов.
- `registries/` - реестры методов, прогонов, источников и репозиториев.
- `projects/` - проектные неймспейсы для координации по отдельным проектам.
- `handoff/` - короткие передачи состояния между сессиями.

## Проектные неймспейсы

- `projects/docspectrum/` - координационный слой `DocSpectrum`: review packets, обсуждения, проектные решения и контракты.

Checker-специфичные файлы, созданные до введения проектных неймспейсов, остаются на текущих местах как legacy-корень.

## Связанные общие папки

- `E:\commons\checker-shared-project` - канонические общие исходные, ссылки на ground truth и справочники.
- `E:\output\checker-shared-project` - результаты общих прогонов и отчеты.
